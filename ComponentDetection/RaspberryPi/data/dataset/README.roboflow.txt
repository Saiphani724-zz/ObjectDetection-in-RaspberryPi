
RaspiDataset - v1 v1
==============================

This dataset was exported via roboflow.ai on November 10, 2020 at 5:43 PM GMT

It includes 1172 images.
A-C-E-G-H-U-T are annotated in YOLO v3 Darknet format.

The following pre-processing was applied to each image:
* Resize to 416x416 (Stretch)

The following augmentation was applied to create 3 versions of each source image:
* 50% probability of horizontal flip
* Random rotation of between -45 and +45 degrees
* Random shear of between -19째 to +19째 horizontally and -21째 to +21째 vertically
* Random Gaussian blur of between 0 and 1.5 pixels
* Salt and pepper noise was applied to 3 percent of pixels
